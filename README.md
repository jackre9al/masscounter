# Mass Gainer Logger — Import chooser (File / Paste)

- **Import ▾** button now opens a tiny menu:
  - **From file…** → file picker (`.csv`)
  - **Paste CSV…** → paste dialog (textarea → Parse & Merge)
- Keeps the minimalist Apple/Google vibe and all previous features:
  - Profiles (email), optional 4‑digit PIN (client‑side SHA‑256)
  - Last 7 days / Last Week / Grand total
  - Auto‑time, custom date/time, edit/delete rows
  - CSV export
- Import de‑duplicates by **ID**, otherwise by **(ISO Timestamp, Grams)**.

Drop the folder into your GitHub Pages repo and hard‑refresh.
